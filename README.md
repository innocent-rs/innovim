# Basic nvim configuration, made from scratch

Features :

- [x] lazy package manager
- [x] telescope
- [x] lsp for lua
- [x] treesiter
- [x] editorconfig (by default)
- [x] cmp
- [x] cattpuccin
- [x] markdown
- [x] basic keybinding for qol
- [x] neotree
- [x] barbar
- [x] git

## Languages supported

- Lua
  - Lint:
  - LSP: lua_ls
  - Format: stylua
- Python
  - Lint: Ruff
  - LSP: Ruff, Pyright
  - Format: Ruff
- HTML/CSS/JavaScript/TypeScript:
  - Lint: eslint
  - lsp: html, css_ls, tailwindcss, emmet_language_server, ts_ls
  - Format: prettierd
- Nix
  - lint:
  - LSP: nil
  - Format:
